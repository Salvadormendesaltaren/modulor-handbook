# Handbook Modulor Studios

> La base de conocimiento del grupo Modulor para potenciar la colaboración, la venta cruzada y el uso de inteligencia artificial.

## Por qué existe este Handbook

Modulor Studios es un grupo de seis boutiques especializadas que operan de forma coordinada: mendesaltaren, Tailor Hub, SSTIL, Minimum, Nocodehackers y FIK. Cada boutique tiene su propio posicionamiento, servicios e identidad, pero comparten clientes, sinergias y una visión común.

Este Handbook nace para resolver un problema concreto: **que cualquier persona del grupo entienda qué hace cada boutique, a quién se dirige y cuándo derivar un cliente o una oportunidad.**

## Para qué sirve

- **Alineación interna.** Conocer la identidad, los servicios y el cliente ideal de cada boutique.
- **Cross-selling.** Saber exactamente cuándo y cómo derivar oportunidades entre boutiques.
- **Contexto para IAs.** Usar estos archivos como base de conocimiento en ChatGPT, Claude u otras herramientas de IA para generar propuestas, emails, presentaciones o análisis con contexto real del grupo.
- **Onboarding.** Que las nuevas incorporaciones tengan una visión completa del ecosistema Modulor desde el primer día.

## Cómo usarlo

Cada capítulo documenta una boutique con su identidad, servicios, cliente ideal, casos de éxito y matriz de derivación. Puedes leerlo linealmente o ir directamente al capítulo que necesites.

Para uso con IAs, descarga los archivos .md desde la sección Repositorio y súbelos como contexto en tu herramienta preferida.

## Versiones del contenido

Esta versión del Handbook (**Team**) contiene información operativa y de posicionamiento. Existe una versión más detallada (**Leadership**) con datos financieros, pricing, equipos completos y estrategia interna, accesible solo por invitación.
